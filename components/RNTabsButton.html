RNTabsButton 标签按钮组件 ​

组件介绍 ​

RNTabsButton 是一个标签按钮组件，基于 Naive UI 的 NTabs 和 NButton 组件封装，提供了标签页和按钮的组合功能，适用于需要在标签页中展示操作按钮的场景。

Props ​

属性名类型默认值说明
modelValue / v-modelstring/number-当前激活的标签值
tabsArray[]标签配置数组
sizestring'medium'尺寸，可选值：'small'/'medium'/'large'
typestring'bar'标签类型，可选值：'bar'/'line'/'card'
justifystring'start'对齐方式，可选值：'start'/'center'/'end'
buttonPositionstring'right'按钮位置，可选值：'left'/'right'
animatedbooleantrue是否开启切换动画
closablebooleanfalse标签是否可关闭

标签配置项 ​

tabs数组中的每个对象应包含以下字段：

typescript
interface TabItem {
  label: string; // 标签显示文本
  value: string | number; // 标签值
  disabled?: boolean; // 是否禁用
  closable?: boolean; // 是否可关闭
  buttons?: Array<{
    // 按钮配置数组
    label: string; // 按钮文本
    type?: string; // 按钮类型
    size?: string; // 按钮尺寸
    disabled?: boolean; // 是否禁用
    onClick?: Function; // 点击事件处理函数
  }>;
}

事件说明 ​

事件名说明回调参数
update:modelValue标签值更新时触发(value: string/number)
tab-click标签被点击时触发(value: string/number)
close标签被关闭时触发(value: string/number)
button-click按钮被点击时触发(buttonConfig: Object, tabValue: string/number)

插槽 ​

插槽名说明
default标签内容区域
prefix标签栏前缀内容
suffix标签栏后缀内容
[tabValue]对应标签值的内容插槽

使用示例 ​

基础用法 ​

vue
<template>
  <rn-tabs-button
    v-model="activeTab"
    :tabs="tabs"
    @button-click="handleButtonClick"
  />
</template>

<script setup>
import { ref } from 'vue';

const activeTab = ref('1');
const tabs = [
  {
    label: '标签1',
    value: '1',
    buttons: [
      { label: '新增', type: 'primary' },
      { label: '删除', type: 'error' },
    ],
  },
  {
    label: '标签2',
    value: '2',
    buttons: [{ label: '编辑', type: 'info' }],
  },
];

const handleButtonClick = (button, tabValue) => {
  console.log('Button clicked:', button, 'in tab:', tabValue);
};
</script>

可关闭标签 ​

vue
<template>
  <rn-tabs-button
    v-model="activeTab"
    :tabs="tabs"
    closable
    @close="handleClose"
  />
</template>

<script setup>
import { ref } from 'vue';

const activeTab = ref('1');
const tabs = ref([
  {
    label: '标签1',
    value: '1',
    buttons: [{ label: '保存', type: 'primary' }],
  },
  {
    label: '标签2',
    value: '2',
    closable: false, // 此标签不可关闭
    buttons: [{ label: '编辑', type: 'info' }],
  },
]);

const handleClose = (tabValue) => {
  const index = tabs.value.findIndex((tab) => tab.value === tabValue);
  if (index !== -1) {
    tabs.value.splice(index, 1);
  }
};
</script>

动态按钮状态 ​

vue
<template>
  <rn-tabs-button v-model="activeTab" :tabs="tabs" />
</template>

<script setup>
import { ref, computed } from 'vue';

const activeTab = ref('1');
const loading = ref(false);

const tabs = computed(() => [
  {
    label: '标签1',
    value: '1',
    buttons: [
      {
        label: loading.value ? '保存中...' : '保存',
        type: 'primary',
        disabled: loading.value,
        onClick: async () => {
          loading.value = true;
          try {
            await saveData();
          } finally {
            loading.value = false;
          }
        },
      },
    ],
  },
]);

const saveData = () => {
  return new Promise((resolve) => setTimeout(resolve, 2000));
};
</script>

自定义按钮位置 ​

vue
<template>
  <rn-tabs-button
    v-model="activeTab"
    :tabs="tabs"
    button-position="left"
    justify="space-between"
  >
    <template #prefix>
      <div class="tab-prefix">
        <icon-custom />
      </div>
    </template>
  </rn-tabs-button>
</template>

<script setup>
const tabs = [
  {
    label: '标签1',
    value: '1',
    buttons: [{ label: '操作1', type: 'primary' }],
  },
];
</script>

<style scoped>
.tab-prefix {
  margin-right: 16px;
}
</style>

<script setup>
const tabs = [

    value: '1',

    ],
  },

  },

注意事项 ​

  1. 每个标签的value必须是唯一的
  2. 按钮的onClick事件会覆盖默认的button-click事件
  3. 标签页切换时会保持按钮的状态
  4. 可以通过buttonPosition控制按钮组的位置
  5. 标签关闭后需要手动更新activeTab的值
  6. 按钮的尺寸默认继承tabs的size属性
  7. 可以通过CSS变量自定义组件的主题样式
